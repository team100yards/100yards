# Enabling GitHub Pages for the Website

GitHub Pages allows you to host static websites directly from a GitHub repository. To enable GitHub Pages for the `team100yards/100yards` repository, follow these steps:

## Step 1: Go to Repository Settings
1. Navigate to your GitHub repository.
2. Click on the `Settings` tab located on the right side of the menu.

## Step 2: Scroll to the GitHub Pages Section
Scroll down to the **GitHub Pages** section in the settings. You will find options to enable GitHub Pages.

## Step 3: Choose a Source
1. Under the **Source** section, select the branch you want to use for GitHub Pages (usually `main` or a `gh-pages` branch).
2. You can also select the folder (usually `/ (root)` or `/docs`) if applicable.

## Step 4: Save the Changes
Click on the `Save` button to enable GitHub Pages.

## Step 5: Access Your Site
Once GitHub Pages is enabled, you will receive a URL to access your site. This URL is usually in the format `https://<username>.github.io/<repository-name>`.

## Notes
- It may take a few minutes for your site to become available after enabling GitHub Pages.
- You can customize your website by adding an `index.html` file or additional HTML files to the selected source folder.
